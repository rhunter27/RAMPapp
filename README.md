## Usage
1. Clone the repository  
   ```bash
   git clone https://github.com/rhunter27/RAMPapp
2. Navigate to the project directory
    * cd RAMPapp/client
3. Install dependencies
    * npm install
4. Run the project
    * npm run dev
5. Open in browser
    * http://localhost:5173/
